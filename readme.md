# 끄투 단어장 (Kkutu Wordbook) 🔍

끄투(KKUTU) 시리즈 게임 플레이를 보조하기 위한 웹 기반 단어 검색 및 전략 도구입니다.  
입력된 글자 수에 따라 **긴 단어**, **공격 단어**, **미션 단어**를 자동으로 구분하여 검색하고, 카드 형태의 UI로 직관적인 결과를 제공합니다.

---

## 📂 프로젝트 구조

```text
📦 kkutu-wordbook
 ├── 📂 long_words/          # 초성별 긴 단어 JSON 파일 저장소
 │    ├── words_ㄱ.json
 │    └── ...
 ├── 📂 templates/           # Flask HTML 템플릿
 │    └── index.html
 ├── 📂 static/              # 정적 파일 (CSS, JS 등)
 ├── app.py                  # 메인 Flask 애플리케이션
 ├── scrape_long.py          # 긴 단어 크롤러
 ├── scrape_mission.py       # 미션 단어 크롤러
 ├── scrape_atk.py           # 공격 단어 크롤러
 ├── atk_words_combined.json # 통합된 공격 단어 DB
 ├── mission_words.json      # 미션 단어 DB
 └── README.md               # 문서
```

## 주요 기능

### 1. 지능형 검색 모드
사용자가 입력하는 검색어의 길이에 따라 자동으로 최적의 검색 알고리즘이 적용됩니다.

- **1글자 입력 시 (예: `가`)**
  - **긴 단어 검색**: 해당 글자로 시작하는 긴 단어(도감용, 점수용)를 `long_words` DB에서 검색합니다.
  - **공격 단어 검색**: 해당 글자로 시작하는 공격 단어를 `atk_words` DB에서 검색하여 길이순으로 정렬해 보여줍니다.

- **2글자 입력 시 (예: `가사`)**
  - **미션 단어 검색**:
    - 첫 글자(`가`)로 시작하면서, 뒷부분에 두 번째 글자(`사`)가 포함된 단어를 찾습니다.
    - 정렬 우선순위: **미션 글자 포함 횟수** (내림차순) > **단어 길이** (내림차순).
  - **Fallback**: 미션 단어 검색 결과가 없을 경우, 자동으로 첫 글자 기준 긴 단어/공격 단어 검색 결과를 보여줍니다.

### 2. 데이터 관리
- **초성 분산 저장**: 긴 단어 데이터는 초성별(`words_ㄱ.json` ~ `words_ㅎ.json`)로 분리되어 있어 빠른 로딩과 검색이 가능합니다.
- **공격 단어 통합**: 끄코위키와 나무위키 등 다양한 출처의 공격 단어를 통합하여 관리합니다.



## 🛠 설치 및 실행 방법

### 1. 프로젝트 클론 및 패키지 설치
```bash
# 필요한 라이브러리 설치
pip install flask requests beautifulsoup4
```

### 2. 데이터 업데이트 (선택 사항)
최신 단어 데이터를 적용하려면 다음 스크립트를 순서대로 실행하세요. (이미 JSON 파일이 있다면 건너뛰어도 됩니다.)

1. **긴 단어 수집**: 끄코위키에서 긴 단어를 크롤링합니다.
   ```bash
   python scrape_long.py
   ```
2. **미션 단어 수집**: 미션 단어를 크롤링하여 `mission_words.json`에 저장합니다.
   ```bash
   python scrape_mission.py
   ```
3. **공격 단어 수집**: 공격 단어를 크롤링하여 `atk_words_kkuko.json`에 저장합니다.
   ```bash
   python scrape_atk.py
   ```
   > *참고: `atk_words_combined.json`은 위 데이터와 나무위키 데이터 등을 합쳐서 별도로 생성해야 합니다.*

### 3. 웹 서버 실행
```bash
python app.py
```
실행 후 브라우저에서 `http://127.0.0.1:5000`으로 접속하세요.

## ✨ 활용 팁
- **미션 처리**: 미션이 '사'일 때, '가'로 시작하는 단어를 찾고 싶다면 `가사`를 입력하세요. '사'가 많이 들어간 긴 단어 순으로 추천해줍니다.
- **공격 방어**: 상대방이 공격했을 때 해당 글자를 '1글자'로 입력하면, 맞받아칠 수 있는 긴 단어나 공격 단어를 빠르게 찾을 수 있습니다.
